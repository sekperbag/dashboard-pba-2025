Panduan Penggunaan
Cara kerja dashboard:
1. Data dibaca dari Google Sheets melalui Apps Script setiap halaman dimuat
2. Auto-refresh setiap 5 menit — atau klik tombol ↻ Refresh di header
3. PIC divisi isi URL Google Drive di tab Kelola Dokumen → klik Simpan
4. Status ✓/✗ dan progress bar terupdate otomatis langsung tanpa reload

⚠️ Link harus URL lengkap:
✓ https://drive.google.com/file/d/xxx/view
✓ https://docs.google.com/spreadsheets/d/xxx
✗ Nama file saja tidak berlaku (mis: Laporan.xlsx)

Embed ke Google Sites:
Insert → Embed → Embed Code → paste tag iframe dari file HTML ini
